# rn-smartkeyboard
iOS inspired cross platform native keyboard with chatGPT embedded, STICKERs, Vogalia etc

#features
- cross-platform
- lightweight
- extensible with tutorials
- more like notepad take any sort of input (taps, gestures, stylus, voice, thumb)
